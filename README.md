# Rakkan

絵にも、本人証明を。

Rakkan は、手描きイラストレーター向けの著作権情報埋め込みツールです。  
イラストに「パスポート」を付与するように著作権情報を与えるコンセプトでデザイン・各機能を開発中です。  
画像ファイル（PNG、JPEG）に著作者情報・利用条件・連絡先などのメタデータを付与することができ、著作権の明示を手軽に行えるような機能を用意しています。  
また、独自の ID システムを用意しています。これを付与すると、手元の画像が一番初めに ID を付与したことが分かりオリジナル画像の証明にも利用できます。  
メタデータが投稿先プラットフォームによって削除されるリスクも鑑み、PNG 形式の画像には上記の情報を透かしで入れるツールも用意しています。
（JPEG 画像は劣化を避けられないため対応予定なし）

## 機能一覧

- **メタデータ付与**：JPEG・PNG を対象に XMP/IPTC 準拠のメタデータを埋め込む
- **プロフィールプリセット**：著作者情報を事前登録しておき、処理時に選択して使える
- **AI 生成画像の検出・リジェクト**：XMP/EXIF/IPTC および C2PA マニフェストを解析し、AI 生成画像への付与を拒否する（手描きイラスト用のアプリとする予定です）
- **LSB 不可視透かし**：PNG のピクセルデータに著作者情報を不可視の透かしとして埋め込む（PNG のみ）
- **著作権証明カード**：処理後にスタンプ付きの証明カードを画像として保存できる

## 技術スタック

- React 19 + TypeScript + Vite
- Tailwind CSS
- ブラウザ標準 API のみでバイナリ処理を実現（外部ライブラリ不使用）
  - `ArrayBuffer` / `Uint8Array`：バイナリ操作
  - `TextEncoder` / `TextDecoder`：文字列 ⇔ バイト変換
  - `crypto.subtle`：SHA-256 ハッシュ生成
  - `CompressionStream` / `DecompressionStream`：zlib 圧縮・展開（LSB 透かし用）

## プライバシー

画像データはすべてブラウザ内で処理される。サーバーへのアップロードは一切行わない。

## 対応ファイル形式

| 形式 | メタデータ付与 | LSB 透かし |
| ---- | -------------- | ---------- |
| JPEG | ✅             | —          |
| PNG  | ✅             | ✅         |

## 埋め込むメタデータ

| 項目                    | タグ                       |
| ----------------------- | -------------------------- |
| 著作者名                | `dc:creator`               |
| 著作権表示              | `dc:rights`                |
| 作品説明・利用許諾案内  | `dc:description`           |
| 作成日（付与日）        | `xmp:CreateDate`           |
| 利用条件・利用許諾案内  | `xmpRights:UsageTerms`     |
| 権利情報 Web サイト URL | `xmpRights:WebStatement`   |
| 連絡先メールアドレス    | `Iptc4xmpCore:CiEmailWork` |
| 連絡先 Web サイト URL   | `Iptc4xmpCore:CiUrlWork`   |
| スタンプ ID（独自）     | `rakkan:StampId`           |

## 未管理著作物裁定制度への対応

著作権法の未管理著作物裁定制度では、利用条件や利用申込先の連絡先が明示されていない場合、著作物が制度の対象となるリスクがあります。

Rakkan では以下の対策を実装しています。

- **利用条件の入力を必須化**：プロフィール登録時に利用条件の入力がなければ保存できない
- **デフォルト値として禁止文言を設定**：特に設定を変更しなくても許可なき転載・複製・改変・二次利用を禁止するメタデータを付与できる
- **利用許諾案内文を自動付加**：連絡先が登録されている場合、`dc:description` と `xmpRights:UsageTerms` に「利用許諾に関するお問い合わせは〇〇までご連絡ください。」を自動で追記する

## ローカル起動

```zsh
pnpm install
pnpm dev
```

## ディレクトリ構成

```
src/
├── types.ts                 # 型定義（CreatorProfile, EmbedMetadata）
├── App.tsx                  # 処理の呼び出し元（パイプラインの起点）
├── components/              # UI コンポーネント
├── hooks/
│   └── useProfile.ts        # プロフィール管理
└── lib/
    ├── xmp-builder.ts       # XMP を文字列として組み立てる
    ├── jpeg-writer.ts       # JPEG バイナリに APP1 セグメントとして挿入
    ├── png-writer.ts        # PNG バイナリに iTXt チャンクとして挿入
    ├── lsb-watermark.ts     # LSB 不可視透かしの埋め込み・読み取り
    ├── metadata-reader.ts   # メタデータ読み取り・AI 検知
    ├── c2pa-reader.ts       # C2PA マニフェスト解析
    ├── ai-detector.ts       # XMP/EXIF/IPTC による AI 生成検知
    └── stamp-id.ts          # スタンプID生成
```
